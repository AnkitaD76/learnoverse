# Landing Page Components

This folder contains the modern landing page and associated components for the Learnoverse edtech platform.

## Components Created

### 1. **Header** (`/components/Header.jsx`)

Modern, responsive navigation header with:

- Logo and brand name
- Desktop navigation menu (Courses, Services, Marketplace, About, Contact)
- Mobile-responsive hamburger menu
- Call-to-action buttons (Login, Get Started)
- Sticky positioning for better UX

### 2. **Footer** (`/components/Footer.jsx`)

Comprehensive footer with:

- Brand section with social media links
- Platform links (Browse Courses, Find Services, Marketplace, etc.)
- Support links (Help Center, About, Contact, FAQ, Blog)
- Legal links (Privacy Policy, Terms of Service, etc.)
- Copyright and language/currency selector

### 3. **Landing Page** (`/pages/Landing/page.jsx`)

Full-featured landing page including:

- **Hero Section**: Eye-catching gradient background with CTA buttons
- **Stats Section**: Key metrics (50K+ learners, 2K+ instructors, etc.)
- **Features Section**: 6 key features with icons
- **How It Works**: 3-step process to get started
- **Testimonials**: Success stories from instructors
- **Final CTA**: Call-to-action to sign up

## Features

### For Course Creators:

- Create and sell courses
- Offer tutoring/consulting services
- Trade educational resources
- Flexible monetization options
- Analytics dashboard
- Secure payments

### Design Elements:

- Modern gradient backgrounds (blue, purple, pink)
- Responsive grid layouts
- Hover effects and transitions

## Color Scheme

- Primary: Blue (#2563EB)
- Secondary: Purple (#9333EA)
- Accent: Pink (#EC4899)
- Background: White/Gray shades
- Text: Gray-900/Gray-600

## Usage

The landing page is now set as the home route (`/`) in the application. Users can:

1. View the landing page at the root URL
2. Click "Get Started" to go to registration
3. Click "Login" to access their account
4. Navigate to various sections via the header menu
